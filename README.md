
# Spare Parts Inventory Demand Forecasting Using Machine Learning

## Project Overview

This project predicts automotive spare parts inventory demand using Machine Learning. The objective is to help service centers optimize inventory levels, reduce inventory costs, and improve spare parts availability through predictive analytics.

## Business Problem

Maintaining spare parts inventory is a major challenge for automotive service centers. Overstocking increases storage costs, while understocking causes service delays and customer dissatisfaction. This project develops a regression model to forecast future spare parts demand.
